Häufig gestellte Fragen
Was ist das hier?
Ernstchan ist ein Imageboard, also ein Forum, in dem hauptsächlich Bilder ausgetauscht werden.
Bitte beachtet, dass ihr hier weder eine Quelle seriöser News, noch einen Kummerkasten finden werdet. Wer nichts mit Sarkasmus und schwarzem Humor anfangen kann, geht am besten gleich wieder.
Wo melde ich mich an?
Auf einem klassischen Imageboard wie Ernstchan kann man sich nicht registrieren. Normalerweise tauscht man sich hier anonym aus.
Wie kann ich in Beiträgen die Schrift formatieren?
Folgende BBCode-Tags stehen zur Auswahl:

Eingabe Ausgabe
[i]kursiv[/i] kursiv
[b]fett[/b] fett
[u]unterstrichen[/u] unterstrichen
[s]durchgestrichen[/s] durchgestrichen
[sp]Spoiler[/sp] Spoiler
[code]Code[/code] Code
(in /c/ mit Syntaxhervorhebung)
Was steckt denn so unter der Haube von Ernstchan?
Ernstchan wird mit Phutaba betrieben, einer in Python (Django) geschriebenen Imageboardsoftware. Unsere Daten befinden sich auf amerikanischen Server.
